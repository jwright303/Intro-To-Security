To run the program enter python3 bloomFilt.py

After the program is executed it will create the bloom filter, then prompt you if you want to enter
the words in from the command line or have them read in from the sample file.
Finally it will print out the word as well as if it thinks its maybe a bad password or a fine password

Note: The packages bitarray, and mmh3 are needed to be able to run this program.
All of these can be found in the requirements.txt file
These can be downlaoded using "pip3 install --user -r requirements.txt"
